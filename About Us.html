MGM Grand

MGM Grand is a program designed to cater the needs of the administration sector of the company. With this program, the admin would be able to work in a more efficient and organized pace in regards to storing the data of their employees and it also features an export system within the program.

What operations can the administrator do within the program?

  • Build an employee list.
  • View the current list of employees.
  • Add a new employee into the system.
  • Remove an existing employee record.
  • Edit an existing employee record.
  • Search an existing employee record.
  • Sort the list of employee records.
  • Export the list of employee records saved into the system.
